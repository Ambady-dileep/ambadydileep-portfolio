# Required assets

Place these files in the `public/` folder before deployment:

| File            | Purpose                                      |
| --------------- | -------------------------------------------- |
| `portfolio.png` | Hero profile photograph (portrait, 3:4 ratio recommended) |
| `AmbadyDileepResume.pdf`    | Resume download button                       |
| `og-image.png`  | Open Graph / social preview (optional)       |

Recommended `portfolio.png` specs: portrait orientation, 3:4 ratio (e.g. 960×1280 or 1200×1600).

Until `portfolio.png` is added, the hero shows an elegant placeholder.
