# Ambady Dileep — Portfolio

Premium product-style personal site — 5 focused sections, scannable in under 90 seconds. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Quick start

```bash
npm install
npm run dev
```

## Before deploying

1. Add your professional portrait as `public/portrait.jpg` (recommended: 800×800px, square crop).
2. Add your resume as `public/resume.pdf`.
3. Optional: add `public/og-image.png` (1200×630) for social sharing previews.

## Deploy to Vercel

```bash
npm run build
```

Connect the repo to Vercel and set the custom domain to `ambadydileep.live`.

## Sections

1. **Hero** — Brand, stats, CTAs, portrait  
2. **Work** — Four featured projects  
3. **Stack** — Frontend, Backend, Database, Tools  
4. **About** — Short statement, philosophy, vision  
5. **Contact** — Email, phone, socials, form  

## Tech stack

- React + Vite
- Tailwind CSS v4
- Framer Motion + Lenis
- Lucide icons (thin stroke)
- Aurora / mesh animated background with cursor spotlight
- React Helmet Async (SEO)
